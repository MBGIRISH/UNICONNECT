## 📱 HOW TO USE EACH FEATURE

### **🔍 SEARCH FOR PEOPLE:**
1. Look at the **top center** of the screen (desktop)
2. Or tap **search icon** 🔍 in header (mobile)
3. **Type a name** → Results appear!
4. **Click person** → View profile
5. **Click message icon** 💬 → Start chat

### **💬 MESSAGE SOMEONE:**
1. **From Search:** Click 💬 next to their name
2. **From Profile:** Click "Message" button
3. **From Header:** Click 💬 to see all conversations
4. **Type message** → Press Enter → Sent!

### **📸 POST WITH IMAGES:**
1. Go to **Feed**
2. Click "What's on your mind?"
3. Type your post
4. Click **image icon** 📷
5. Select image(s)
6. Click "Post"

### **❤️ LIKE & COMMENT:**
1. See a post you like
2. Click **heart icon** ❤️ to like
3. Click **comment icon** 💬 to comment
4. Type comment → Press Enter
5. See it appear in real-time!

### **📅 CREATE AN EVENT:**
1. Go to **Events**
2. Click "Create Event"
3. Fill in details
4. Upload cover image
5. Select category
6. Click "Create"

### **👥 JOIN A GROUP:**
1. Go to **Groups**
2. Click "Join Group" on any group
3. Or click "Create New Group"
4. Chat with group members!

### **🛒 SELL AN ITEM:**
1. Go to **Marketplace**
2. Click "Sell Item"
3. Upload images (up to 5)
4. Set price, title, description
5. Click "List Item"

### **💰 BUY AN ITEM:**
1. Browse **Marketplace**
2. Click item to view details
3. Click "Message Seller"
4. Chat about price, meetup
5. Arrange to meet & complete deal!

```

└── socialLinks

posts/{postId}
├── authorId, content, imageUrls
├── likesCount, commentsCount
└── subcollections:
    ├── likes/{userId}
    └── comments/{commentId}

events/{eventId}
├── title, description, location
├── startTime, endTime, coverImage
└── subcollections:
    └── attendees/{userId}

groups/{groupId}
├── name, description, isPrivate
└── subcollections:
    ├── members/{userId}
    └── messages/{messageId}

marketplace/{listingId}
├── title, price, images
├── sellerId, condition
└── category

messages/{messageId}
├── text, imageUrl
├── senderId, receiverId
└── createdAt, read
```
