# 🔍 WHERE TO FIND "RESOURCE TYPE" IN CLOUDINARY

## ❌ **NOT IN "Advanced" Tab**

The "Resource type" setting is **NOT** in the Advanced tab you're currently viewing.

---

## ✅ **WHERE TO FIND IT:**

### **Option 1: "General" Tab** (Most Common Location)

1. **Click "General"** in the left sidebar (first item with gear icon)
2. **Look for:** "Resource type" dropdown or checkboxes
3. **Should show options like:**
   - `Auto` (recommended - allows all types)
   - `Image`
   - `Video`
   - `Raw` (for documents/PDFs)
   - Or checkboxes to enable each type

---

### **Option 2: "Transform" Tab**

1. **Click "Transform"** in the left sidebar (second item)
2. **Scroll down** to find "Resource type" setting
3. **Set to:** `Auto` or enable Image, Video, Raw separately

---

### **Option 3: Main Upload Settings Page**

If not in tabs, it might be at the top of the preset editor:
1. **Scroll to the very top** of the page
2. **Look for** "Resource type" dropdown near the preset name

---

## 🎯 **WHAT TO SET:**

**Best Option:** Set to **`Auto`** (allows all file types automatically)

**OR** enable these checkboxes:
- ✅ **Image** (for photos)
- ✅ **Video** (for videos)
- ✅ **Raw** (for documents/PDFs)

---

## 📋 **STEP-BY-STEP:**

1. **Click "General" tab** (gear icon, first in sidebar)
2. **Look for "Resource type"** dropdown or section
3. **Set to:** `Auto` OR enable Image, Video, Raw
4. **Click "Save"** (top right)

---

## ⚠️ **IF YOU CAN'T FIND IT:**

Some Cloudinary accounts have it in different places:
- Try "Transform" tab
- Try scrolling to top of "General" tab
- It might be called "Allowed resource types" or "File types"

---

## ✅ **QUICK CHECK:**

After setting Resource type:
- ✅ Should allow image uploads
- ✅ Should allow video uploads  
- ✅ Should allow document/PDF uploads

If you still can't find it, take a screenshot of the "General" tab and I'll help locate it!
