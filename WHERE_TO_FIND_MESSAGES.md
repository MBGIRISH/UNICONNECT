## 💡 **PRO TIPS:**

### **Can't Find Messages?**
1. Look at **TOP RIGHT** of any page
2. Find the 💬 icon
3. If it has a red badge, you have unread messages!
4. Click it → Opens Messages page

### **Want to Message Someone?**
1. Go to their profile (search their name)
2. Click blue **"Message"** button
3. Chat opens automatically

### **Check All Conversations:**
1. Click 💬 icon (header)
2. See full list of chats
3. Click any to open

## 🔍 **COMMON QUESTIONS:**

### **Q: Where is the Messages page?**
**A:** Click the 💬 icon in the top right corner of any page!

### **Q: How do I know if I have new messages?**
**A:** Red badge appears on the 💬 icon with the count (e.g., "3")

### **Q: Where do I see all my conversations?**
**A:** Messages page → Left sidebar (desktop) or full screen (mobile)

### **Q: How do I start a new conversation?**
**A:** Go to someone's profile → Click "Message" button → Start chatting!

### **Q: Can I see old messages?**
**A:** Yes! Open the conversation and scroll up to see message history
