✅ **Study Groups** - Chat with classmates + AI help  
✅ **Marketplace** - Buy/sell items

### **Step 3: Try the NEW Features!**

#### **🔍 SEARCH FOR PEOPLE:**

**Desktop:**
1. Look at the **TOP CENTER** of the screen
2. See the search bar: 🔍 Search people...
3. Type any name (e.g., "John")
4. Results appear below!
5. Click person → View profile
6. Click 💬 → Send message

**Mobile:**
1. Tap **search icon** 🔍 in top right
2. Full-screen search opens
3. Type a name
4. Tap person to view profile
5. Tap 💬 to message

---

#### **💬 MESSAGE SOMEONE:**

**Method 1: From Search**
1. Search for a person
2. Click the 💬 icon next to their name
3. Chat opens → Type "Hello!" → Press Enter
4. ✅ Message sent!

**Method 2: From Profile**
1. Click any person's name anywhere
2. See their profile
3. Click **"Message"** button
4. Chat opens → Start chatting!

**Method 3: View All Chats**
1. Click **message icon** 💬 in top right header
2. See all your conversations
3. Click any conversation
4. Continue chatting!

## 🎮 TRY THESE FEATURES

### **1. Create a Post**
1. Go to **Feed**
2. Type in "What's on your mind?"
3. Click 📷 to add image (optional)
4. Click **Post**
5. ✅ Posted!

---

### **2. Like & Comment**
1. See a post in Feed
2. Click ❤️ to like (heart turns red!)
3. Click 💬 to comment
4. Type comment → Press Enter
5. ✅ Comment appears in real-time!

---

### **3. Create an Event**
1. Go to **Events**
2. Click **Create Event**
3. Fill details + upload cover image
4. Select category (Academic/Social/etc.)
5. Click **Create**
6. ✅ Event created!

---

### **4. RSVP to Event**
1. See any event
2. Click **⭐ Interested**
3. Star turns yellow!
4. Attendee count goes up!
5. ✅ You're going!

---

### **5. Join a Study Group**
1. Go to **Groups**
2. Click **Join Group** on any group
3. Or click **Create New Group**
4. Chat with members!
5. Share images!
6. Try **@AI** for study help!

---

### **6. Sell an Item**
1. Go to **Marketplace**
2. Click **Sell Item**
3. Upload up to 5 images
4. Set price, title, description
5. Click **List Item**
6. ✅ Item listed!

---

### **7. Buy an Item**
1. Browse **Marketplace**
2. Click any item
3. View details & images
4. Click **Message Seller**
5. Chat about price & meetup
6. ✅ Deal made!

---

### **8. Search & Message** (NEW!)
1. Look at **top of screen**
2. Type in search bar: "John"
3. Click 💬 next to their name
4. Type "Hi!"
5. Press Enter
6. ✅ Message sent!

---

### **9. Edit Your Profile**
1. Go to **Profile**
2. Click **Edit Profile**
3. Upload avatar
4. Add bio, location, phone
5. Add social links (Twitter, LinkedIn, etc.)
6. Click **Save**
7. ✅ Profile updated!

## 🎯 COMMON QUESTIONS

### **Q: Where is the search bar?**
**A:** Top center on desktop, tap 🔍 on mobile (top right)

### **Q: How do I message someone?**
**A:** Search for them → Click 💬 icon OR Go to their profile → Click "Message"

### **Q: Where do I see my messages?**
**A:** Click 💬 icon in top right header

### **Q: Can I send images in chat?**
**A:** Yes! Click 📷 in the chat input

### **Q: How do I like a post?**
**A:** Click the ❤️ icon on any post

### **Q: How do I comment?**
**A:** Click 💬 on a post, type, press Enter

### **Q: How do I RSVP to events?**
**A:** Click ⭐ Interested on any event

### **Q: How do I upload images?**
**A:** Click 📷 icon anywhere (posts, events, marketplace, messages)

### **Q: Is everything free?**
**A:** Yes! Using Firebase free tier + Cloudinary free tier

### **Q: Does it work on mobile?**
**A:** Yes! Fully responsive design

## 🔥 PRO TIPS

### **Search Tips:**
- Type just 2 letters to start searching
- Results update as you type
- Click person → profile
- Click 💬 → instant message

### **Messaging Tips:**
- All conversations in Messages page
- Unread counts show in badges
- Send images via 📷 button
- Messages are real-time (instant!)

### **Marketplace Tips:**
- Always message seller first
- Meet in public places
- Inspect item before paying
- Use "Mark as Sold" when done

### **Group Chat Tips:**
- Share study materials via images
- Use @AI for study help
- Create private groups for close friends
- Public groups for campus-wide

### **Profile Tips:**
- Upload a clear avatar
- Write a good bio
- Add social links for networking
- Share your profile link
