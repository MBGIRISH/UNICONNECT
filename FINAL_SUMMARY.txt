================================================================================
  🚨 IMPORTANT: WHAT YOU NEED TO DO NEXT (10 MINUTES)
================================================================================

Your app is showing an authentication error because Firebase services
need to be enabled. This is NORMAL and takes only 10 minutes to fix.

📖 FOLLOW THIS GUIDE: Open "NEXT_STEPS.md"

Or follow these quick steps:

1️⃣  Enable Email/Password Auth (2 min)
    → https://console.firebase.google.com/project/uni-connect-b63b0/authentication/providers
    → Click "Email/Password" → Enable → Save
    ✅ This fixes your current error!

2️⃣  Enable Firestore Database (3 min)
    → https://console.firebase.google.com/project/uni-connect-b63b0/firestore
    → Create Database → Test mode → Enable

3️⃣  Deploy Security Rules (1 min)
    → In Firestore, go to "Rules" tab
    → Copy contents from firestore.rules file
    → Paste and Publish

4️⃣  Enable Cloud Storage (2 min)
    → https://console.firebase.google.com/project/uni-connect-b63b0/storage
    → Get Started → Test mode → Done

5️⃣  Deploy Storage Rules (1 min)
    → In Storage, go to "Rules" tab
    → Copy contents from storage.rules file
    → Paste and Publish

6️⃣  Initialize Sample Data (30 sec)
    → In terminal: npm run init-db

7️⃣  Refresh Browser
    → Hard refresh: Cmd+Shift+R or Ctrl+Shift+R
    → Error should be GONE!
