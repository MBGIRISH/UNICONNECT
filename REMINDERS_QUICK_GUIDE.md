## 🎨 **COLOR GUIDE:**

### **Amber (Yellow-Orange):**
- **Meaning:** Class starting soon (within 15 min)
- **Banner:** Amber background
- **Badge:** "Soon"
- **Action:** Get ready, prepare to leave

### **Green:**
- **Meaning:** Class in progress (right now)
- **Banner:** Green background
- **Badge:** "Live" with pulsing dot
- **Action:** Class is ongoing

### **Gray:**
- **Meaning:** Normal, no alerts
- **Banner:** None
- **Badge:** None
- **Action:** Relax, class is later

## 💡 **QUICK TIPS:**

### **See "Soon" Badge?**
- ✅ Class starts within 15 minutes
- ✅ Time to head to class!
- ✅ Check location/room number

### **See "Live" Badge?**
- ✅ Class is happening now
- ✅ Check end time
- ✅ Useful if you're running late

### **No Badges?**
- ✅ All clear!
- ✅ Next class is > 15 min away
- ✅ Or no classes today
