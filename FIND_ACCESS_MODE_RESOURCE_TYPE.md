# 🔍 Where to Find "Access Mode" and "Resource Type"

## 📍 EXACT LOCATIONS

### **Option 1: In "General" Tab (Scroll Down)**

You're currently on the "General" tab. These settings are **below** what you can see:

1. **Scroll down** past "Generated display name" section
2. **Keep scrolling** - these settings are further down
3. **Look for:**
   - **"Access mode"** or **"Delivery type"** section
   - **"Resource type"** or **"Allowed resource types"** section

---

### **Option 2: Check "Optimize and Deliver" Tab**

1. **Look at the top** of the preset editor page
2. **See tabs:** General | Transform | Manage and Analyze | **Optimize and Deliver** | Advanced
3. **Click:** **"Optimize and Deliver"** tab
4. **Look for:**
   - **"Access mode"** or **"Delivery type"**
   - **"Resource type"** settings

---

### **Option 3: Check "Advanced" Tab**

1. **Click:** **"Advanced"** tab (rightmost tab)
2. **Look for:**
   - **"Access mode"**
   - **"Resource type"**
   - **"File type restrictions"**

---

## 🎯 STEP-BY-STEP SEARCH

### **Step 1: Scroll Down in "General" Tab**

1. **On the current page** (General tab)
2. **Use mouse wheel** or **scroll bar** to scroll down
3. **Look for sections** that come after "Generated display name"
4. **You should see:**
   - "Access mode" dropdown
   - "Resource type" dropdown
   - "File size limits"
   - Other settings

### **Step 2: If Not Found, Check Other Tabs**

**Click each tab and look:**

1. **"Transform"** tab → Usually for image transformations
2. **"Manage and Analyze"** tab → Usually for metadata
3. **"Optimize and Deliver"** tab → **CHECK HERE!** ⭐
   - This tab often has "Access mode" and delivery settings
4. **"Advanced"** tab → **CHECK HERE TOO!** ⭐
   - This tab often has resource type and file restrictions

---

## 🔍 WHAT TO LOOK FOR

### **"Access Mode" Might Be Called:**
- "Access mode"
- "Delivery type"
- "Access control"
- "Public/Private"
- "Asset access"

### **"Resource Type" Might Be Called:**
- "Resource type"
- "Allowed resource types"
- "File type"
- "Asset type"
- "Upload type"

---

## 💡 QUICK CHECK

### **If You Can't Find Them:**

**Don't worry!** Your preset might already be set correctly:

1. **"Unsigned" signing mode** = Usually means "Public" access by default
2. **Code specifies resource_type** = So preset doesn't need it
3. **Just click "Save"** - it should work!

---

## ✅ ALTERNATIVE: Check Default Settings

If you can't find these in the preset:

1. **Go to:** Settings → Upload (main settings, not preset)
2. **Look for:** Default access mode and resource type
3. **These apply to all presets** unless overridden

---

## 🎯 RECOMMENDED ACTION

1. **First:** Scroll down in "General" tab (keep scrolling!)
2. **If not found:** Click "Optimize and Deliver" tab
3. **If still not found:** Click "Advanced" tab
4. **If still not found:** Just click "Save" - your current settings should work!

**The most important setting (Unsigned) is already correct! ✅**

---

## 📸 WHAT TO LOOK FOR VISUALLY

### **Access Mode:**
- Usually a **dropdown** with options like:
  - "Public"
  - "Private"
  - "Authenticated"
- Should be set to **"Public"**

### **Resource Type:**
- Usually a **dropdown** with options like:
  - "Auto"
  - "Image"
  - "Video"
  - "Raw"
- Should be set to **"Auto"** (if available)

---

**Try scrolling down first, then check the other tabs!**
